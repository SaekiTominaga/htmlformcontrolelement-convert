`FormControlConvert.js` Demo

Remove whitespace at both ends

e.g. ␣hoge␣ → hoge

delete blank lines

e.g. hoge⏎⏎piyo → hoge⏎piyo

Make alphanumeric characters half-width

e.g. ｈｏｇｅ－１２３ → hoge-123

Make alphanumeric characters full-width

e.g. hoge-123 → ｈｏｇｅ－１２３

Make full-width space half-width

Consolidate contiguous spaces

e.g. hoge␣␣piyo → hoge␣piyo

Make the alphabet lowercase

e.g. HOGE → hoge

Make the alphabet uppercase

e.g. hoge → HOGE

Proprietary conversion table

e.g. 50．0％ → 50.0% (When { '．': '.', '％': '%' } is specified as the second argument)
